Detailed Description

Klasa zarządzająca głównym interfejsem użytkownika.

  • MainWindow pełni rolę kontrolera pośredniczącego między interfejsem (UI), klientem sztucznej inteligencji (OllamaClient) oraz warstwą trwałości danych (DatabaseManager).
